MarkDown的简单使用

  Markdown是一种流行的标记语言，通常为程序员所使用，但是它实际上是个非常简单、非常
容易学习的语法。接下来，就简单的介绍下Mardown。后面还有丰富的Mardown的编辑器提供给大家参考。

段落和换行

  Mardown以一段完整的空行作为段落间隔。

标题

  Mardown在每一行的首部插入1-6个 # 符号，对应到标题的1到6阶， # 符号和标题之间需要一个空格，例如：

# 这是一级标题

## 这是二级标题

### 这是三级标题

#### 这是四级标题

##### 这是五级标题

###### 这是六级标题

如下图所示：

区块引用

  Markdown标记区块引用是使用类似e-mail中用 > 的引用方式，在需要引用的文字之前使用 > 符号即可，注意在 > 符号与引文之间需要一个空格。

  当需要进行区块引用，也就是多段引用时，只需要在每段之前都加上 > 符号，包括空行。

  引用允许嵌套使用，引文内的引文只需要连续使用 > 符号即可。

  引文内任何Markdown语法依旧生效，比如列表、标题等。例如：

> 这是引文的第一段

> 这是引文的第二段

>> 这是内嵌的二级引文

列表

  Mardown支持非常直观的无序列表和有序列表。

无序列表

  无序列表使用 + 加号， - 减号和 * 星号加空格来标识，列表的嵌套使用 tab 制表符缩进，例如：

这是一个无序列表

+ 项目1

+ 项目2

+ 项目3

这是一个无序列表

- 项目1

- 项目2

  -项目2.1

  -项目2.2

- 项目3

这是一个无序列表

* 项目1

* 项目2

* 项目3

  * 项目3.1

    * 项目3.1.1

如下图所示：

有序列表

  当然有序列表也非常直观，只需要使用数字接着一个 . 英文句点然后空格，就可以标识一个有序列表了。在这里任何一个数字都可以触发有序列表，并按正确的数序显示。例如：

这是一个有序列表

1. 项目一

2. 项目二

3. 项目三

这也是一个有序列表

8. 项目一

1. 项目二

2. 项目三

强调

  Mardown使用 * 星号和 _ 下划线作为标记强调字段的符号，被一个 * 或 _ 包围的字词会被显示为 斜体 ，用两个 * 或 _ 包围起来的话，则会被显示为 粗体 ，用三个 * 或 _ 包围起来的话则会被显示为 粗斜体 ，例如：

_我是一行斜体字_

**我是一行粗体字**

***我是一行粗斜体***

代码标识和代码块

  作为一名IT人员，有时要往自己的文稿中写入大量的代码，则Mardown可以很好的实现，并且代码区块中的所有输入都会按照原样输出，包括连续的空格。

  Mardown中代码标识主要有行内代码和代码区块两种。行内代码用一对`反引号` 标识。例如：

我是一个 `Alt` 键，当我设置变量 `int a = 20;`时，就可以内嵌行代码了。

插入链接

   Mardown支持两种形式的链接语法，行内式和参考式。
现在我就介绍下行内式语法。

行内式的语法如下：

[链接文字](链接地址)

[链接文字](链接地址 “title属性” )

  其中(链接地址)\如果是绝对地址记住加上http://,title 属性是指鼠标移到链接上显示的文字。

插入图片

  插入图片的语法和插入链接类似，只需在链接的语法前面加一个!感叹号就可以了，如下所示：

![链接文字](链接地址)

![链接文字](链接地址 “title属性” )

编辑工具

在线的Mardown编辑器

  1. dillinger 漂亮强大，支持md, html, pdf 文件导出。支持dropbox, onedrive，google drive, github. 来自国外，可能不够稳定。

  2. MaHua 小众软件推荐，界面有些简陋。

  3. 简书 一个很不错的博客平台，每几秒钟便会自动存入一个备份。可以直接从本地拖入照片生成链接，一直在不断优化。作为一个博客平台，需要注册账号后方能进行写作。

Windows平台编辑器

  1. MarkdownPad 一款全功能的编辑器，被很多人称赞为windows 平台最好用的markdown编辑器.

  2. Miu 一款模仿mou 的windows平台markdown编辑器，小众推荐，必属精品，官网无法打开，好在小众提供了百度云下载。界面美观，功能不够成熟.

其他的一些编辑器大家可以上月光博客浏览，我这里只介绍了一部分。
